# CIFAR-10 Classification with PyTorch

This repository contains a unified Python script for training and evaluating various Convolutional Neural Network (CNN) and baseline models on the CIFAR-10 dataset.

The project uses the following key components:

### Prerequisites

1. **Python Environment:** Python 3.8+ installed.
2. **Requirements:** latest torch, torchvision
3. Training command.

   ```bash
   python main.py --mode train --model_type simple_cnn --exp_name simple_cnn_run_1
4. Testing command.
 ```bash
   python main.py --mode test --model_type simple_cnn --exp_name simple_cnn_run_1
